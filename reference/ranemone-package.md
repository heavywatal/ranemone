# ranemone: R Interface to ANEMONE DB

ANEMONE DB is a database for biodiversity surveys based on environmental
DNA metabarcoding. It is managed by Dr. Michio Kondoh, Ecological
Integration Lab. at Tohoku University. This package provides an utility
to analyze data from ANEMONE DB.
